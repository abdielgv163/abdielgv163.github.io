Proyecto
Google Maps Scraper

Herramienta desarrollada para facilitarme la recolección de datos desde Google Maps, esto con la finalidad de generar DataFrames de forma automatizada para mis proyectos personales de Data Science.
La información que se puede recopilar con esta herramienta son: el nombre del lugar, URL, dirección, estrellas, total de reviews, sitio web, teléfono y horarios.

Este proyecto ha sido realizado con la ayuda de librerías como pandas, parsel y selenium.
