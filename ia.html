Proyecto
Predictor de rating de libros

Análisis completo del set de datos GoodReads, que cuenta con miles de libros con información relevante, con la finalidad de entender los datos y posteriormente crear un modelo que nos permita predecir el rating de libros utilizando un algoritmo de regresión.

"Titanic"
Machine Learning from Disaster

Solución a la competencia "Titanic" de Kaggle utilizando árboles de decisión, obteniendo un accuracy de 0.758. El proyecto está documentado en Inglés desde en análisis exploratorio, limpieza de datos, procesamiento y creación del modelo.
